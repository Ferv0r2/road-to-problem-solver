# [스택](https://www.algodale.com/data-structures/stack/)

- LIFO (Last In First Out) / 입출구가 하나인 구조
- 배열 구현:
  - 스택의 크기가 미리 정해져 있다.
  - 들어가는 데이터 개수의 변동 폭이 적을 때 (즉, 크기 가늠이 가능할 때)
  - 정적 할당
- 리스트 구현:
  - 연결 리스트의 특성상 스택 크기가 자유롭다.
  - 들어가는 데이터 개수의 가늠이 어려울 때
  - 동적 할당
  - 단일 연결 리스트
