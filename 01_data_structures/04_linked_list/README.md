# [링크드 리스트](https://www.algodale.com/data-structures/linked-list/)

- Node/Link
  - Node: 실제 정보를 담고 있는 데이터
  - Link: 노드 간의 연결을 나타내는 포인터
- 특징
  - 동적인 자료 구조
  - 순차적 접근 가능한 자료 구조
  - 불연속 메모리 사용 (물리적 순서 X)
- 구현
  - 단일 연결 리스트 (단방향: data, next pointer)
  - 이중 연결 리스트 (양방향: data, next pointer, prev pointer)
